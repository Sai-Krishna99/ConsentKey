# ConsentKey Windows Actions SDK Wrapper

This folder is reserved for the Logitech toolkit-generated Windows wrapper project.

Purpose:
- keep Windows-only Logitech runtime files out of the shared cross-platform source
- let the team use one repo while isolating generated SDK artifacts
- provide a stable place for local device testing with MX Creative Console and Actions Ring

What belongs here:
- the project generated by `npx @logitech/plugin-toolkit create consentkey-actions-windows`
- Logitech SDK dependencies
- local Windows-only build outputs and link artifacts

What does not belong here:
- product docs
- shared broker logic
- the canonical plugin source you want teammates to edit cross-platform

Expected workflow:
1. Generate the wrapper project here on Windows.
2. Copy or sync shared source from `../consentkey-actions/src/` into the wrapper project's `src/`.
3. Run the Logitech watch/build commands from this folder on Windows.
4. Test the plugin against the Next.js broker running from the root repo.

Notes:
- This folder is intentionally documented in git, but common generated artifacts inside it are ignored by `.gitignore`.
- If the Logitech toolkit creates additional generated folders, add them to the repo `.gitignore` rather than moving the shared source out of this repo.
